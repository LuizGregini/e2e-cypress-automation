# 📱 E2E Cypress Automation

Projeto de automação de testes End-to-End (E2E) utilizando **Cypress + JavaScript**, com foco em boas práticas, organização e simulação de fluxos reais de usuário.

---

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de dar os primeiros passos na automação de testes. Durante o processo de estudo e pesquisa, o **Cypress** foi escolhido por ser uma ferramenta de fácil aprendizado e amplamente utilizada no mercado.

Abaixo estão as principais abordagens e práticas exploradas durante esse período de aprendizado:
- Praticar automação de testes E2E
- Simular o comportamento real do usuário na aplicação
- Consolidar conhecimentos em testes automatizados com Cypress
- Criar uma base reutilizável e escalável para automação web

---

## 🚀 Tecnologias utilizadas

- JavaScript
- Cypress
- Node.js
- Mocha (test runner padrão do Cypress)

---

## 🧪 O que foi automatizado

Os testes cobrem fluxos reais da aplicação, como:

- ✅ Acesso à aplicação
- ✅ Interação com elementos da interface
- ✅ Validação de comportamentos esperados
- ✅ Execução de fluxos completos (E2E)

---

## 📂 Estrutura do projeto

```bash
cypress/
 ├── e2e/           # Testes end-to-end
 ├── fixtures/      # Massa de dados (mock)
 ├── support/       # Commands customizados e configurações globais
cypress.config.js   # Configuração do Cypress
package.json        # Dependências do projeto
```
